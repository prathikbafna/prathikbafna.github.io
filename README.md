# prathikbafna.github.io

Static portfolio site (Option A: plain HTML/CSS/JS) designed for GitHub Pages.

## Local run
You can open `index.html` directly, or run a tiny local server:

```bash
python -m http.server 8000
```

Then open http://localhost:8000

## Customize
- Update links (LinkedIn, email, GitHub) in `index.html` and role pages.
- Update projects in `assets/data/projects.json`
- Drop resumes into `assets/resume/` and update the buttons.

## Deploy
GitHub repo must be named: `prathikbafna.github.io`

Settings → Pages → Deploy from branch → `main` / root
